# Texteditor

## Version 0.1

Ein einfacher Texteditor mit grundlegenden Formatierungsfunktionen und Darkmode.

### Aktuelle Funktionen:

- Texteingabe und -bearbeitung
- Schriftgrößenänderung
- Schriftartauswahl
- Schriftfarbenauswahl
- Textformatierung (Fett, Kursiv, Unterstrichen)
- Neues Dkomument erstellen (Abfrage ob bestehender Text gespeichert werden soll)
- Speichern und Öffnen von .rtf Dateien ( damit die Formatierung beibehalten bliebt wurde .rtf genommen)
- Darkmode (umschaltbar)
- Statusmeldungen beim Speichern und Laden

### Geplante Funktionen für zukünftige Versionen:

- Listenfunktion
- Textausrichtung (links, zentriert, rechts)
- Export als PDF
